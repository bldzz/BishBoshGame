Bish-Bosh 2.0

A fun number game! Enter your custom values below and see how it works.

Bish-Bosh lists all numbers between 1 and 100, but some exceptions: If the number is evenly divisible by 3, ​`Bish' ​is listed instead of the number. If the number is evenly divisible by 4, ``Bosh'' is listed instead of the number. If the number is evenly divisible by both 3 and 4, “​Bish-Bosh​” is written instead of the number.
